1. What is JQuery?

jquery created by John Resig in 2006.

jQuery is a fast, small, and feature-rich JavaScript library.

jQuery makes a web developer's life easy.

jQuery is easy to learn because it supports same JavaScript style coding.

jQuery provides excellent online API documentation.

jQuery provides excellent cross-browser support without writing extra code.

It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

jQuery provides Selectors to retrieve DOM element based on different criteria like tag name, id, css class name, attribute name, value, nth child in hierarchy etc.

2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in JQuery, JQuery Animation?

1. How to apply css using jquery :-

There are two types of apply css using jquery :-

(A) one of which is used to change a single CSS property :-

this method only takes 2 parameters, namely the property name and the value.

Syntax : $().css(propertyname, value);

(B) The other is used to change multiple CSS properties simultaneously :-

With the second method, you can pass a JSON string object as a parameter that you want to apply to the HTML element. JSON string objects contain CSS properties along with their values.

Syntax : $().css(properties);


2. How to Add Class and Remove Class in JQuery :-

Syntax :-

  • Add Class :-

    • $('selector').addClass(class_name);

  • Remove Class :- :-

    • $('selector').removeClass(class_name);


    3. JQuery Animation :-

    The .animate() method allows us to create animation effects on any numeric CSS property. The only required parameter is a plain object of CSS properties. This object is similar to the one that can be sent to the .css() method, except that the range of properties is more restrictive.

    we can produce various types of animation using the animate() method in jQuery.

    This method can produce simple to complex animation in the web page.

    Using animation, we can change the properties of HTML elements such as background colour, changing border styles, changing navigation properties, formatting the font properties, etc.
